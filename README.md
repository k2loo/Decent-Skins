# Decent-Skins
Light mode is digital nudity, maintain your modesty with Decent Skins for your rack modules.

## How to use
1. Download the zip file from the downloads folder and unzip it.
2. Copy/paste the Decent Skins svg files into the `res` folder for each brand you wish to reskin.
3. Close VCV Rack if you had it open, and reopen it.

## FAQ

Q: Where is my plugins folder?
  
A: Locate it by opening VCV Rack and right clicking on a module > info > open plugin folder. [plugin folder location](/Raw%20Files/z_img/plugin-folder-location.png)

Q: How do I revert to the old skins?

A: There are two methods. First, you could download the original backup files in this repo and replace the Decent Skins svg files with the originals. Second, you could unsubscribe from your modules for a moment and then resubscribe to them. In any case, do this with the Rack app closed - it will only notice file changes when it opens again.
