# Decent Skins for VCV Rack
Rid yourself of those pesky light mode modules in your rack with Decent Skins. Thus far fiftyish modules from half a dozen or so brands have a more sensible color scheme offered here.

![decent skins thumb](/Raw%20Files/z_img/thumb-1000x503.png)

[![ko-fi](/Raw%20Files/z_img/ko-fi-button-branded.svg)](https://ko-fi.com/B0B21PMGFU)

## How to use
1. Download the zip file from the downloads folder and unzip it.
    -  Close VCV Rack if it's running.

2. Copy/paste the Decent Skins svg files into the `res` folder for each brand's modules you wish to reskin. The new files should have the same filenames as the originals.
    - Find it by opening VCV Rack and right clicking on a module > info > open plugin folder, [like this.](/Raw%20Files/z_img/plugin-folder-location.png) Once found, close VCV Rack and paste the files.

3. Open VCV Rack and enjoy.

## FAQ

*Why are the screws in my modules still light?*

A: That's baked into the plugin code for many modules, the files provided here are only background images.

*Q: How do I revert to the default skins?*

A: There are two methods.

1. You could replace the Decent Skins svg files with the originals, also available in this repo.
2. You could unsubscribe from your modules for a moment and then resubscribe to them.

In any case, closing the VCV Rack app and reopening it will cause it to read the plugin folders and notice any changes.

---

Disclaimer: If a module gets updated by it's maker, you may have to replace the default background files with Decent Skins files again.
